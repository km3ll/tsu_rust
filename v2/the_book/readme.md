## commands

### `cargo`

```bash
# project
cargo clean
cargo fmt
cargo build

# target/debug
cargo run
cargo run --bin base_package
cargo run --bin another_main
# target/release
cargo run --release

# projects
cargo new package_name # binary crate
cargo new --lib package_name # library crate
cargo install cargo-modules
cargo --version
```

### `docker`

```bash
docker build .
```
