## commands

### `cargo`

```bash
# project
cargo clean
cargo fmt
cargo build

# target/debug
cargo run
# target/release
cargo run --release

cargo run --bin the_book
cargo run --bin app

# projects
cargo new package_name # binary crate
cargo new --lib package_name # library crate
cargo install cargo-modules
cargo install cargo-make
cargo --version
```
